## 架构分层

系统从上到下分为交互层、通信层、服务中心、基础设施和数据层。

- 交互层：由 React 和 React Router DOM 构成，负责页面展示、路由组织和用户操作入口。
- 通信层：负责实时口试通信，包括 WebRTC、pipeline、TTS 和 STT。
- 服务中心：包含口试模块和管理模块。口试模块负责打分规则、出题规则、口试状态维护和口试文件管理；管理模块负责课程管理、考试管理、口试记录管理和角色管理。
- 基础设施：提供 Agents、Tools 和全程监控能力。
- 数据层：使用 Mysql 和 Meilisearch 存储业务数据与检索数据。
